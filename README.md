# College Memories Project

A website to preserve and share memories from our college journey (Batch 2023-26).

## Features
- Photo Gallery
- Member Profiles
- Memory Notes/Thoughts
- Timeline View

## Tech Stack
- **Frontend:** React + TypeScript
- **Backend:** Node.js + Express
- **Database:** MongoDB

## Getting Started

### Frontend
```bash
cd Frontend
npm install
npm start
```

### Backend
```bash
cd Backend
npm install
npm start
```
